# This app is based on Python3 Interpreter

# Install the requirements
  Run Command `pip3 install requirements.txt`

# Setup the database
  Run Command `flask db init` to initialize the database
              `flask db migrate` to create migration files folder
              `flask db upgrade` to commit the migration change

# add some catalogs for the app
  Since CRUD operations is for the items not the catalogs, we add some catalogs by the script
  Run the python file `python3 add_catalogs.py`

# Run the web app
  Run the python file `python3 app.py`
